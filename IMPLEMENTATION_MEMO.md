## 2. イベントステージ
### riemu
- 敵として riemu が登場
- フレンドと一緒にクリアを目指す（共闘）
- フレンドは1体だけキャラを選んで参加
- riemu を倒したら riemu をゲット
- **riemu レアリティ（レベル別）**
  - 1〜66: コモン
  - 67〜133: レア
  - 134〜200: スーパーレア
  - 201〜266: ウルトラレア
  - 267〜399: レジェンド
  - 400: HST
  - （ガチャのレアリティのみ使用、STARY なし）

### tetsuya
- 敵として tetsuya が登場
- tetsuya を倒したら tetsuya をゲット
- **tetsuya レアリティ（レベル別）**：riemu と同様
  - 1〜66: コモン
  - 67〜133: レア
  - 134〜200: スーパーレア
  - 201〜266: ウルトラレア
  - 267〜399: レジェンド
  - 400: HST
- **tetsuya スキル**：相手の攻撃力を下げる（2ターン）

## 3. イベントガチャ
- **meah** を排出
- **smile** を排出

## 4. meah スキル
- 2ターン無敵
- 全体の攻撃力を2倍にするバフ
